# 种子计数器 - Seed Counter (OpenCV.js)

这是一个基于纯浏览器端 (WebAssembly) 的种子自动计数工具。专为农业科研人员设计，用于快速统计种子数量，特别针对十字花科（如小白菜、青梗菜）等微小种子进行了算法优化。

## 🌟 核心特性

- **纯前端处理**：图片无需上传服务器，保护科研数据隐私，响应速度快。
- **粘连分离算法**：采用距离变换 (Distance Transform) 逻辑，有效解决种子堆叠、靠拢导致的计数偏少问题。
- **抗阴影干扰**：集成自适应阈值 (Adaptive Thresholding) 算法，在光照不均或有阴影的实验环境下依然精准。
- **多维度过滤**：支持基于“面积”和“圆度”的双重过滤，自动剔除树枝、灰尘等杂质。
- **全平台支持**：适配手机浏览器，支持实验室现场拍照识别。

## 🔬 适用场景

- 种子千粒重 (Thousand-kernel Weight) 测定。
- 蔬菜品种表型分析中的数量统计。
- 种子发芽率实验前期计数。

## 🛠️ 参数调节指南

- **分离强度**：数值越高，对粘连种子的切割越强。若种子被切得太碎，请调低此值。
- **最小圆度**：用于过滤长条状杂质。纯圆种子的值接近 100%。
- **形态学核**：用于消除图片上的细小噪点。
